Real Estate landing page

Develop a web page for a real estate that will serve to show the properties for sale and for rent. It also contains a contact form to communicate with a real estate agent. The technologies I used were HTML, CSS, JavaScript and PHP.

02. Mexico City Juarez mission inventory

Inventory of houses for furniture control

I developed a web application with which we could keep track of the furniture in more than 50 houses. The application connects to a database which stores the amount of furniture that the house has, this was used every time the house changed guests. If the data entered is different from what you had previously, send a message to the administrator. The technologies used were HMTL, CSS, JavaScript, PHP, MySql.
